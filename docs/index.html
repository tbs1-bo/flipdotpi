Das Projekt dient der Ansteuerung einer Flipdot-Zielanzeige mit verschiedenen Mikrocontrollern oder einem RaspberryPi. Das folgende Bild zeigt eine solche Anzeige.

Kleine runde Plättchen mit unterschiedlichen Farben auf den Seiten sind so gelagert, dass ein Strom genutzt werden kann, um sie durch einen Magneten kippen zu lassen. Damit kann dann ein Bild oder Text auf dem Display dargestellt werden.
